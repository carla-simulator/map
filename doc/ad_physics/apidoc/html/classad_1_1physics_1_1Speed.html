Detailed Description

DataType Speed.

The rate of change of an object's position with respect to time. The speed of an object is the magnitude of its velocity. The unit is: MeterPerSecond

◆ restrictToLimitsAndEnsureValid()

void ad::physics::Speed::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the Speed to it's defined limits

If the Speed isNormal(), but exceeds the defined limits, it is restricted to its limits. If Speed::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
