◆ restrictToLimitsAndEnsureValid()

void ad::physics::DistanceSquared::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the DistanceSquared to it's defined limits

If the DistanceSquared isNormal(), but exceeds the defined limits, it is restricted to its limits. If DistanceSquared::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
