◆ restrictToLimitsAndEnsureValid()

void ad::physics::Distance::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the Distance to it's defined limits

If the Distance isNormal(), but exceeds the defined limits, it is restricted to its limits. If Distance::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
