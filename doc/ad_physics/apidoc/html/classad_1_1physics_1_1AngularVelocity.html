Detailed Description

DataType AngularVelocity.

The rate of change of angular velocity of an object with respect to time. The unit is: RadianPerSecond

◆ restrictToLimitsAndEnsureValid()

void ad::physics::AngularVelocity::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the AngularVelocity to it's defined limits

If the AngularVelocity isNormal(), but exceeds the defined limits, it is restricted to its limits. If AngularVelocity::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
