Detailed Description

DataType AngularAcceleration.

The rate of change of AngularVelocity of an object with respect to time. The unit is: RadianPerSecondSquared

◆ restrictToLimitsAndEnsureValid()

void ad::physics::AngularAcceleration::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the AngularAcceleration to it's defined limits

If the AngularAcceleration isNormal(), but exceeds the defined limits, it is restricted to its limits. If AngularAcceleration::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
