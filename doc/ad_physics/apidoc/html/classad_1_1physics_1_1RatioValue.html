◆ restrictToLimitsAndEnsureValid()

void ad::physics::RatioValue::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the RatioValue to it's defined limits

If the RatioValue isNormal(), but exceeds the defined limits, it is restricted to its limits. If RatioValue::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
