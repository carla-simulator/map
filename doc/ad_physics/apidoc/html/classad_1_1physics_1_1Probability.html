◆ restrictToLimitsAndEnsureValid()

void ad::physics::Probability::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the Probability to it's defined limits

If the Probability isNormal(), but exceeds the defined limits, it is restricted to its limits. If Probability::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
