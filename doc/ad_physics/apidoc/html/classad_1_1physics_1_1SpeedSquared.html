◆ restrictToLimitsAndEnsureValid()

void ad::physics::SpeedSquared::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the SpeedSquared to it's defined limits

If the SpeedSquared isNormal(), but exceeds the defined limits, it is restricted to its limits. If SpeedSquared::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
