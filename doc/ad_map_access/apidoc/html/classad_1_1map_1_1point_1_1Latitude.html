◆ restrictToLimitsAndEnsureValid()

void ad::map::point::Latitude::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the Latitude to it's defined limits

If the Latitude isNormal(), but exceeds the defined limits, it is restricted to its limits. If Latitude::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
