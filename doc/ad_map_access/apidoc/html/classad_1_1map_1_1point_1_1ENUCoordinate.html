◆ restrictToLimitsAndEnsureValid()

void ad::map::point::ENUCoordinate::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the ENUCoordinate to it's defined limits

If the ENUCoordinate isNormal(), but exceeds the defined limits, it is restricted to its limits. If ENUCoordinate::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
