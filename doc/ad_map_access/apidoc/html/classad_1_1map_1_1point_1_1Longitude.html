◆ restrictToLimitsAndEnsureValid()

void ad::map::point::Longitude::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the Longitude to it's defined limits

If the Longitude isNormal(), but exceeds the defined limits, it is restricted to its limits. If Longitude::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
