◆ restrictToLimitsAndEnsureValid()

void ad::map::point::Altitude::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the Altitude to it's defined limits

If the Altitude isNormal(), but exceeds the defined limits, it is restricted to its limits. If Altitude::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
