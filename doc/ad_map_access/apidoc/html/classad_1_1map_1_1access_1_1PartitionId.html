◆ restrictToLimitsAndEnsureValid()

void ad::map::access::PartitionId::restrictToLimitsAndEnsureValid ( )
inline

if possible restrict the PartitionId to it's defined limits

If the PartitionId isNormal(), but exceeds the defined limits, it is restricted to its limits. If PartitionId::isNormal() returns false an std::out_of_range() exception is thrown.

  • not isNormal(): std::out_of_range() exception is thrown
  • cMinValue <= value <= cMaxValue: nothing is done
  • value < cMinValue: resulting value = cMinValue
  • value > cMaxValue: resulting value = cMaxValue
